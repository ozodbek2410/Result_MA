# Улучшения для мобильных устройств

## Исправленные проблемы

### 1. TestEditor - Редактор вопросов
**Было:** Варианты ответов располагались в одну строку, создавая бардак на маленьких экранах
**Стало:** 
- Каждый вариант в отдельном блоке с вертикальной раскладкой
- Буква варианта и кнопки действий в шапке
- Текстовое поле на всю ширину
- Изображения адаптивны (max-width: 100%)

### 2. RichTextEditor - Текстовый редактор
**Улучшения:**
- Адаптивные отступы (меньше на мобильных)
- Скрытие подсказок на маленьких экранах
- Уменьшенный размер шрифта на мобильных (14px → 16px на desktop)
- Горизонтальная прокрутка для длинных формул

### 3. FormulaPopover - Редактор формул
**Улучшения:**
- Попап занимает почти весь экран на мобильных (calc(100vw - 16px))
- Компактные вкладки (только иконки на мобильных)
- Сетка символов 6 колонок вместо 8 на мобильных
- Уменьшенные отступы и размеры кнопок
- Адаптивное позиционирование (центрирование на мобильных)
- Уменьшенная высота редактора формул

### 4. CSS стили редактора
**Добавлено:**
- Медиа-запросы для адаптивных размеров
- word-wrap и overflow-wrap для длинных текстов
- overflow-x: auto для формул
- Адаптивные min-height для полей ввода

## Рекомендации по тестированию

Проверьте на устройствах с шириной экрана:
- 320px (iPhone SE)
- 375px (iPhone 12/13)
- 390px (iPhone 14)
- 414px (iPhone Plus)
- 768px (iPad)
