# Настройка фонового изображения в Word

## Автоматическая настройка (Рекомендуется)

### PDF
PDF экспорт автоматически добавляет фоновое изображение `/logo.png` с прозрачностью 95%.

### Word
Запустите скрипт для автоматического добавления водяного знака:

```bash
cd server
npm run add-watermark
```

Скрипт:
1. Создаст резервную копию `reference.backup.docx`
2. Добавит `logo.png` как водяной знак в `reference.docx`
3. Настроит прозрачность 50%

После этого все экспортированные Word документы будут содержать фоновое изображение.

## Ручная настройка (Альтернатива)

Если автоматический скрипт не сработал, настройте вручную:

### Шаг 1: Открыть reference.docx

Откройте файл `server/templates/reference.docx` в Microsoft Word.

### Шаг 2: Добавить водяной знак

1. Перейдите в **Design** (Дизайн) → **Watermark** (Водяной знак)
2. Выберите **Custom Watermark** (Настраиваемый водяной знак)
3. Выберите **Picture watermark** (Рисунок)
4. Нажмите **Select Picture** и выберите `client/public/logo.png`
5. Установите **Scale** (Масштаб): 100% или Auto
6. Снимите галочку **Washout** (Обесцветить) если хотите цветное изображение
7. Нажмите **OK**

### Шаг 3: Настроить прозрачность (опционально)

Для более тонкой настройки прозрачности:

1. Вставьте изображение через **Insert** → **Pictures**
2. Щелкните правой кнопкой → **Wrap Text** → **Behind Text**
3. Настройте размер и положение
4. Щелкните правой кнопкой → **Format Picture** → **Picture** → **Transparency**: 95%

### Шаг 4: Сохранить

Сохраните `reference.docx` и закройте Word.

## Проверка

После настройки все экспортированные Word документы будут содержать фоновое изображение.

## Восстановление

Если что-то пошло не так, восстановите из резервной копии:

```bash
cd server/templates
copy reference.backup.docx reference.docx
```

## Технические детали

Скрипт работает с Open XML форматом Word:
- Распаковывает DOCX (это ZIP архив)
- Добавляет изображение в `word/media/watermark.png`
- Создает `header1.xml` с водяным знаком
- Обновляет связи в `document.xml.rels`
- Запаковывает обратно в DOCX
